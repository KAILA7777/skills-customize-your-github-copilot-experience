# 📘 Assignment: File-Based To-Do List

## 🎯 Objective

Practice file input and output in Python by building a simple to-do list that saves tasks between runs.

## 📝 Tasks

### 🛠️ Create the To-Do List Program

#### Description
Write a Python program that lets a user add tasks to a to-do list and display them on request.

#### Requirements
Completed program should:

- Start with a menu that allows the user to view tasks or add a new task.
- Store tasks in a text file so they remain available after the program closes.
- Display all saved tasks when the user chooses to view them.
- Keep the program simple and easy to follow.

### 🛠️ Save and Load Tasks

#### Description
Improve the program so it loads tasks from a file when it starts and writes updates back to the file.

#### Requirements
Completed program should:

- Read tasks from a file when the program begins.
- Add new tasks to the list and save them to the file.
- Show a friendly message if the file does not exist yet.
- Preserve previously saved tasks when the program is run again.

### 🛠️ Add Completion Tracking

#### Description
Extend the program so users can mark tasks as completed.

#### Requirements
Completed program should:

- Let the user mark a task as done.
- Update the stored task information in the file.
- Display completed tasks clearly.
